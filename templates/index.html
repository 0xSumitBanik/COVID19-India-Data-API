COVID19 India API

Country Data

In the Country Data API, you'll get the details of passengers screened count, active cases, recovered cases, migrated cases and death cases.

Try it out.

States and UT Data

In the States and UT Data API, you'll get the details of confirmed cases count, cured cases and death cases.

Try it out.

ICMR Lab Details API

In the ICMR Lab Details API, you'll get the name and addresses of all the Government Labs, Private Labs approved by ICMR for testing.

Try it out.

India Helpline API

In the India Helpline API, you'll get the list of all the Helpline Numbers of the States and the Union Territories.

Try it out.
